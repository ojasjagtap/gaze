Nothing happens when I click Start:

  • Check that you're on a regular web page (not chrome:// pages)
  • Refresh this page
  • Check browser console (F12) for errors
  • Make sure camera permission is granted

Calibration doesn't appear:

  • Check camera permissions (click camera icon in address bar)
  • Make sure webcam is working
  • Reload the extension

Page doesn't scroll:

  • Make sure you're looking near the edges (top 30% or bottom 30%)
  • Keep your gaze steady
  • Try increasing sensitivity
  • Press Ctrl+Shift+G to see debug info

üí° Performance Notes

The extension aims for 20-30 frames per second for gaze estimation. You can monitor performance using the debug panel (Ctrl+Shift+G).

Expected metrics:

  • FPS: 20-30 (higher is better)
  • Velocity: Changes based on where you look
  • Confidence: Should be > 0.6 for scrolling
  • Fixation: Shows "Yes" when gaze is stable
